Alerts - Alertas

Proporcione mensajes de respuesta contextual para acciones típicas del usuario con el puñado de mensajes de alerta disponibles y flexibles.

Examples - Ejemplos

Las alertas están disponibles para cualquier longitud de texto, así como un botón de cierre opcional. Para un estilo adecuado, utilice una de las ocho clases contextuales requeridas (por ejemplo, .alert-success). Para el cierre en línea, utilice el complemento JavaScript de alertas.

¡Atención! A partir de la versión 5.3.0, el mixin Sass alert-variant() está obsoleto. Las variantes de alerta ahora tienen sus variables CSS anuladas en un bucle Sass.

Dismissing - Desestimar

Utilizando el plugin JavaScript de alertas, es posible descartar cualquier alerta en línea. He aquí cómo:

  • Asegúrate de haber cargado el plugin de alertas, o el JavaScript compilado de Bootstrap.
  • Añade un botón de cierre y la clase .alert-dismissible, que añade padding extra a la derecha de la alerta y posiciona el botón de cierre.
  • En el botón de cierre, añada el atributo data-bs-dismiss="alert", que activa la funcionalidad JavaScript. Asegúrate de utilizar el elemento button con él para que se comporte correctamente en todos los dispositivos.
  • Para animar las alertas al descartarlas, asegúrese de añadir las clases .fade y .show.

Cuando se descarta una alerta, el elemento se elimina completamente de la estructura de la página. Si un usuario de teclado descarta la alerta utilizando el botón de cierre, su foco se perderá repentinamente y, dependiendo del navegador, se restablecerá al inicio de la página/documento. Por esta razón, recomendamos incluir JavaScript adicional que escuche el evento closed.bs.alert y establezca programáticamente focus() en la ubicación más apropiada de la página. Si está planeando mover el foco a un elemento no interactivo que normalmente no recibe foco, asegúrese de añadir tabindex="-1" al elemento.
