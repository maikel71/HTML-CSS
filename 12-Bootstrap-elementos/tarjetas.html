Cards - Tarjetas

Las tarjetas de Bootstrap proporcionan un contenedor de contenido flexible y extensible con múltiples variantes y opciones.

About - Acerca de

Una card(tarjeta) es un contenedor de contenidos flexible y ampliable. Incluye opciones para encabezados y pies de página, una amplia variedad de contenidos, colores de fondo contextuales y potentes opciones de visualización. Si estás familiarizado con Bootstrap 3, las tarjetas sustituyen a nuestros antiguos panels(paneles), wells(pozos) y thumbnails(miniaturas). Funcionalidad similar a esos componentes está disponible como clases modificadoras para las tarjetas.

Example - Ejemplo

Las tarjetas se construyen con la menor cantidad posible de marcas y estilos, pero aún así se las arreglan para ofrecer una tonelada de control y personalización. Construidas con flexbox, ofrecen una alineación sencilla y combinan bien con otros componentes de Bootstrap. No tienen margin por defecto, así que usa utilidades de espaciado según sea necesario.

A continuación se muestra un ejemplo de una tarjeta básica con contenido mixto y una anchura fija. Las tarjetas no tienen anchura fija al principio, por lo que ocuparán naturalmente toda la anchura de su elemento padre. Esto se puede personalizar fácilmente con nuestras distintas opciones de tamaño.

Content types - Tipos de contenido

Las tarjetas admiten una amplia variedad de contenidos, como imágenes, texto, grupos de listas, enlaces y mucho más. A continuación se muestran ejemplos de lo que se admite.

Body - Cuerpo

El bloque de construcción de una tarjeta es el .card-body. Utilízalo siempre que necesites una sección acolchada dentro de una tarjeta.

Se trata de un texto dentro del cuerpo de una tarjeta.

Titles, text, and links - Títulos, texto y enlaces

Los títulos de las tarjetas se utilizan añadiendo .card-title a una etiqueta h*. Del mismo modo, los enlaces se añaden y colocan uno al lado del otro añadiendo .card-link a una etiqueta a.

Los subtítulos se utilizan añadiendo un .card-subtitle a una etiqueta h*. Si los elementos .card-title y .card-subtitle se colocan en un elemento .card-body, el título y el subtítulo de la tarjeta quedan bien alineados.

Sizing - Dimensionado

Las tarjetas no asumen un width(ancho) específico para empezar, por lo que serán 100% anchas a menos que se indique lo contrario. Puedes cambiar esto según sea necesario con CSS personalizado, clases de cuadrícula, mixins Sass de cuadrícula o utilidades.

Images - Imágenes

Las tarjetas incluyen varias opciones para trabajar con imágenes. Puedes añadir "image caps" al final de la tarjeta, superponer imágenes al contenido de la tarjeta o simplemente incrustar la imagen en la tarjeta.

Image caps - Tapas de imagen

Al igual que las cabeceras y los pies de página, las tarjetas pueden incluir "image caps" superiores e inferiores, es decir, imágenes en la parte superior o inferior de una tarjeta.

Card layout - Diseño de tarjetas

Además de estilizar el contenido de las tarjetas, Bootstrap incluye algunas opciones para diseñar series de tarjetas. Por el momento, estas opciones de diseño aún no son responsivas.

Card groups - Grupos de tarjetas

Utilice los grupos de tarjetas para mostrar las tarjetas como un único elemento adjunto con columnas de igual anchura y altura. Los grupos de tarjetas comienzan apilados y utilizan display: flex; para unirse con dimensiones uniformes a partir del punto de interrupción sm.

Si quieres tener este tipo de diseño en la v5, puedes hacer uso del plugin Masonry. Masonry no está incluido en Bootstrap, pero hemos creado un ejemplo de demostración para ayudarte a empezar.
