Documentacion de Bootstrap

Modal

Utiliza el plugin modal JavaScript de Bootstrap para añadir cuadros de diálogo a tu sitio para lightboxes, notificaciones al usuario o contenido completamente personalizado.

How it works - Cómo funciona

Antes de empezar con el componente modal de Bootstrap, asegúrate de leer lo siguiente, ya que nuestras opciones de menú han cambiado recientemente.

  • Los modales se construyen con HTML, CSS y JavaScript. Se sitúan por encima de todo lo demás en el documento y eliminan el desplazamiento del (body) para que el contenido del modal se desplace en su lugar.
  • Al hacer clic en el "telón de fondo" del modal, éste se cerrará automáticamente.
  • Bootstrap sólo admite una ventana modal a la vez. Los modales anidados no están soportados ya que creemos que son malas experiencias de usuario.
  • Los modales utilizan position: fixed, que a veces puede ser un poco particular sobre su representación. Siempre que sea posible, coloca el HTML modal en una posición de nivel superior para evitar posibles interferencias de otros elementos. Es probable que te encuentres con problemas al anidar un .modal dentro de otro elemento fijo.
  • Una vez más, debido a position: fixed, hay algunas advertencias con el uso de modals en dispositivos móviles. Consulta la documentación de compatibilidad con navegadores para obtener más información.
  • Debido a cómo HTML5 define su semántica, el atributo HTML autofocus no tiene efecto en los modals de Bootstrap. Para conseguir el mismo efecto, utilice JavaScript personalizado:

    • const myModal = document.getElementById('myModal') const myInput = document.getElementById('myInput') myModal.addEventListener('shown.bs.modal', () => { myInput.focus() }).

El efecto de animación de este componente depende de la consulta de medios prefers-reduced-motion.
