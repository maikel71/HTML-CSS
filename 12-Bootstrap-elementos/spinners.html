Spinners - Hilanderos

Indique el estado de carga de un componente o página con los spinners de Bootstrap, creados íntegramente con HTML, CSS y sin JavaScript.

About - Acerca de

Los "spinners" de Bootstrap pueden utilizarse para mostrar el estado de carga en tus proyectos. Están construidos sólo con HTML y CSS, lo que significa que no necesitas JavaScript para crearlos. Sin embargo, necesitarás JavaScript personalizado para cambiar su visibilidad. Su apariencia, alineación y tamaño pueden personalizarse fácilmente con nuestras increíbles clases de utilidad.

Por motivos de accesibilidad, cada cargador incluye aquí role="status" y un span class="visually-hidden">Loading... /span> anidado.

El efecto de animación de este componente depende de la consulta de medios prefers-reduced-motion.

Alignment - Alineación

Los Spinners en Bootstrap están construidos con rems, currentColor, y display: inline-flex. Esto significa que se pueden redimensionar fácilmente, cambiar de color y alinear rápidamente.
