Images - Imágenes

Documentación y ejemplos para que las imágenes tengan un comportamiento responsive (de modo que nunca sean más anchas que su padre) y añadirles estilos ligeros, todo mediante clases.

Responsive images - Imágenes receptivas

Las imágenes en Bootstrap se hacen responsive con .img-fluid. Esto aplica max-width: 100%; y height: auto; a la imagen para que se adapte al ancho de la imagen principal.

Image thumbnails

Además de nuestras utilidades border-radius, puede utilizar .img-thumbnail para dar a una imagen un aspecto de borde redondeado de 1px.

Aligning images

Alinee las imágenes con las clases flotantes auxiliares o las clases de alineación de texto. las imágenes a nivel de bloque se pueden centrar utilizando la clase de utilidad de margen .mx-auto.
